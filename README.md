## Descripción breve

MyTriboo es una pequeña plataforma de red social orientada a la gestión de usuarios e intereses. Permite que los usuarios creen y compartan publicaciones y eventos en tablones, gestionen conexiones (pendientes, aceptadas, bloqueadas) y se comuniquen mediante chats privados y grupales. La implementación en este repositorio cubre modelos para usuarios, intereses, publicaciones, eventos, mensajes y un gestor de base de datos en Java.

Para usos de desarrollo y ejecución local se mantiene la siguiente nota técnica:

```markdown
Para el correcto funcionamiento del Main con los assert es necesario activar "-ea" en las opciones VM del Run/Debug Configuration (en IntelliJ)

```
